About FieldKit

Built with care for offline-first productivity

Giorgiy Shepov
Creator

Passionate about building tools that work offline, with privacy at the core. FieldKit is a collection of productivity and learning apps designed to empower users to stay focused and productive, anywhere.

🎯 Mission

To create a comprehensive suite of offline-first tools that respect user privacy while providing powerful productivity, learning, and field capabilities. FieldKit empowers individuals to work anywhere — on airplanes, in remote areas, or simply without relying on constant internet connectivity.

💡 Philosophy

Privacy First: All data stays on your device. No tracking, no analytics, no third-party services.

Offline by Default: Core functionality works completely offline. Network features are optional.

Simple & Fast: Clean interfaces, minimal bloat. Each tool does one thing well.

Open & Transparent: Built with open-source technologies. Code is available on GitHub.

🏗️ Technology

Backend: Go 1.22+ with embedded static assets and optional API endpoints

Frontend: Vanilla HTML, CSS, and JavaScript — no frameworks, no bloat

Distribution: Available as native binaries for macOS, Linux, and Windows

PWA: Works as a Progressive Web App with offline service worker support
