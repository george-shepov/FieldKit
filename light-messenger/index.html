Field Guide

Protocol used in this build:

- Start marker: -.-.- - Payload: UPPERCASE MESSAGE + '=' + CRC8 hex - End marker: ...-.- - Repeat count: configurable (default 3) - Dot: 1 unit on, Dash: 3 units on - Intra-symbol gap: 1 unit off - Letter gap: 3 units off - Word gap: 7 units off

Recommendations: - Use short messages under 40 characters. - Start with 120-140 ms unit speed. - Keep devices steady and line-of-sight. - In bright daylight, increase threshold and reduce distance. - Torch mode may not be supported by every phone/browser.
